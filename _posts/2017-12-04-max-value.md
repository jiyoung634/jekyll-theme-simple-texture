# 최대값(Max) 구하기

```java
	public static void main(String[] args) {
		// double 자료형 배열 선언 및 초기화
		double[] myList = { 1.9, 2.9, 3.4, 3.5 };

		
		// Finding the largest element
		double max = myList[0];
		for (int i = 1; i < myList.length; i++) {
			if (myList[i] > max)
				max = myList[i];
		}
		System.out.println("Max is " + max); 
	}
}
```
